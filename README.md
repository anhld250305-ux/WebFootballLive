# WebFootballLive

WebFootballLive là một ứng dụng web Single Page Application (SPA) cao cấp chuyên trực tiếp bóng đá HD, cập nhật lịch thi đấu, bảng xếp hạng và tỉ số trực tuyến thời gian thực với phong cách thiết kế thể thao neon và hiệu ứng 3D sống động.

## 🚀 Tính năng nổi bật
* **Quả bóng 3D tương tác (Cyber Football):** Tích hợp Three.js (React Three Fiber) tạo hiệu ứng 3D xoay mượt mà theo thời gian và di chuyển khi cuộn trang hoặc di chuột.
* **Chất lượng phát trực tiếp HD:** Luồng video HD 1080p, 60fps mượt mà, độ trễ cực thấp.
* **Bình luận Tiếng Việt & Tiếng Anh:** Đội ngũ bình luận viên năng động, vui vẻ.
* **Thiết kế Glassmorphism hiện đại:** Giao diện tối kết hợp với các hiệu ứng kính mờ (glassmorphic) neon xanh cỏ và vàng gold.
* **Hỗ trợ đa ngôn ngữ:** Dễ dàng chuyển đổi giữa Tiếng Việt và Tiếng Anh.

## 💻 Công nghệ sử dụng
* **Frontend Core:** React 19, Vite 8, JavaScript
* **3D Graphics:** Three.js, `@react-three/fiber`, `@react-three/drei`
* **Animations:** Framer Motion
* **Routing:** React Router Dom v7
* **Localization:** i18next & react-i18next
* **Styling:** Vanilla CSS (Glassmorphism & Neon theme)

## 🛠️ Hướng dẫn cài đặt & Chạy cục bộ

1. Cài đặt các thư viện phụ thuộc:
   ```bash
   npm install
   ```

2. Khởi chạy máy chủ phát triển local:
   ```bash
   npm run dev
   ```

3. Mở trình duyệt và truy cập: [http://localhost:5173/](http://localhost:5173/)
